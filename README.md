 # Sri-Perfumy

Sri-Perfumy is a simple perfume product website built using HTML and CSS. It displays perfume products in a clean card layout with an attractive design for users to explore different perfumes.

## Features
- Clean perfume product layout
- Attractive product cards
- Simple and modern UI
- Easy navigation
- Responsive design

## Technologies Used
- HTML
- CSS
